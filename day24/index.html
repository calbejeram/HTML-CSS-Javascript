Adding HTML DOM Elements

  1. Document.createElement(element);

    -adds element to the DOM.

  2. document.createTextNode(text in quotes);

    -this creates the text that can go inside and HTML element.

  3. parentElement.appenChild(childElement);

  4. parentElement.insertBefore(newElement, existingElement);

  5. parentElement.removeChild(childElement);

    -removes the childElement

  6. parentElement.replaceChild(newElement, oldElement);

    -new element will replace the old element specified.

3 Ways to manipulate content of an element

  1. .innerHTML - it returns all text including HTML tags that is contained by an element.
  2. .innerText - it returns all text contained by an element and all its child element.
  3. .textContent - is all text contained by an element and all its children that are for formatting purpose only.
