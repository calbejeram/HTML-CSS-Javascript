CSS Position Property

It indicate the position of elements in an HTML Documents.

position: relative; - it means that it follows the regular flow of the document and the ofsets relative to itself according to the properties top, right, bottom, and left.

This is a div tag with a relative position

position: fixed; - an element is relative to the browser window or the viewport

This is a div tag with a fixed position

position: sticky; - sets an element as relative however it becomes fixed when the element reached a specific position during scrolling

This is a footer tag with a sticky position

position: absolute; - this element is eliminated from the regular flow of the document and other element are ignored
