Creating Containers with Bootstrap

    3 Different types of containers

  1. fixed - this is the .container class that provides responsive a responsive fixed width container.
  2. fluid - this is the .container-fluid that provides a full width container spanning
  3. responsive - allow you to specify class that is 100% wide until the specified breakpoint is reached. The breakpoints can be
    • sm
    • md
    • xl
    • xxl
