CSS Gradients

There are three types of CSS gradients

  1. linear-gadient : create a linear gradients that shows you a multiple colors within a straight line.
  2. radial-gradient : creates images that transition color which radiate in specific spot

CSS Colors

There are four values types for CSS Colors

  1. Color keyword : setting colors for html elements
  2. RGB values : A CSS function that has a parameter values which are red, green, and blue. ex. rgb(red, green, blue).
  3. Hexadecimal value : values that starts with "#".
  4. HSL: stands for Hue, Saturation, and Lightness. ex. (hsl(Hue, Saturation, Lightness))

CSS Typography

Popular CSS text styling options

These are the following Css properties
  1. color : sets the color of a text
  2. text-align : is for the alignment of html text
  3. text-transform : transforms the text in uppercase, lowercase or capitalize
  4. text-decoration : sets decoratve lines to emphasize points of a text
  5. text-indent : is use for indentation of the first line of each paragraph
  6. letter-spacing : is used for spacing between characters
  7. word-spacing : is used for spacing between words
  8. line-height : is used for spacing vertically between lines of text
  9. direction : is used for defining the direction of a text, It can be form left to the right of vice versa
  10. text-shadow : applies a shadow to the text
  11. text-overflow : defines how overflow text of the contect
  12. overflow-wrap : breaks long word and wrap onto the next line. Used to prevent text from overflowing from the line box.
  13. word-break : sets the line breaking rules when text overflow in its content box
